### What I learned

I learned how to position an absolute element in the middle of its parent. I took me a while to figure it out but in the end am happy I tackled this challenge

```css

.profile__info {
    position: absolute;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    margin: auto;
    display: flex;
    justify-content: center;
    align-items: center;
}
```
